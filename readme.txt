Описание:
eReboot - это инструмент для автоматической очистки временных файлов на вашем компьютере при перезагрузки.

Основные функции:
1. Очистка временных файлов.
2. Очистка корзины.
3. Очистка кэша 1С.

Инструкции по использованию:
1. Запустите приложение и ждите перезагрузки компьютера.

Примечания:
- При возникновении проблем обратитесь к автору или проверьте наличие обновлений.
